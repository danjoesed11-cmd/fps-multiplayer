Gameplay

Six Game Modes

From classic team battles to asymmetric hide-and-seek — every session plays differently.

⚔️

Team Deathmatch

Two teams fight for the kill lead. First to 30 eliminations wins the match.

🚩

Capture the Flag

Steal the enemy flag and return it to your base. First team to 3 captures wins.

🏆

Zone Wars

Control capture zones to bleed points from your opponents. Hold more zones, win faster.

🫥

Hide & Seek / Prop Hunt

Hiders blend into the map. Seekers hunt them down before the clock runs out.

💀

Team Wipeout

Round-based elimination. No respawns per round — win 3 rounds to take the match.

🤖

Singleplayer

Fight against AI bots across all maps. Full economy and upgrades available.

Systems

Everything Connected

Each system is built to work together — economy feeds weapons, kills feed economy, modes feed the lobby.

🌐

Real-Time Multiplayer

ENet-powered networking with server-authoritative hit validation, MultiplayerSynchronizer for movement, and reliable RPCs for game events.

💰

Economy & Shop

Earn coins per kill and objective. Visit buy zones to purchase new weapons or upgrade existing ones across three tiers.

💬

Lobby Chat

Full server-relayed text chat in the lobby with channel support. Ready-up system and mode/map voting before each match.

🎨

Character Customization

Equip cosmetic skins for body, head, gloves, and boots. Kill effects unlock with coins. All changes replicate to every player.

🗺️

Map System

Maps define spawn points, objective markers, and buy zones. Game modes plug in as child nodes — swap modes without reloading maps.

📡

Event Bus Architecture

All cross-system signals route through a central EventBus singleton. HUD, economy, and modes stay fully decoupled.

Code

Clean GDScript Architecture

Server-authoritative networking. Every coin, kill, and flag capture validated server-side.

# EconomyManager.gd — server validates every purchase
@rpc("any_peer", "call_local", "reliable")
func request_purchase(item_id: String) -> void:
    if not multiplayer.is_server(): return
    var sender_id := multiplayer.get_remote_sender_id()
    _process_purchase(sender_id, item_id)

# WeaponBase.gd — client claims hit, server confirms
@rpc("any_peer", "call_local", "reliable")
func rpc_claim_hit(victim_id: int, hit_pos: Vector3) -> void:
    if not multiplayer.is_server(): return
    # server raycasts to confirm line of sight
    victim.take_damage(final_damage, sender_id, weapon_id)
